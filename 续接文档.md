## 环境要求
- Python 3.7+（使用了is_relative_to的兼容性处理）
- Flask 3.0.0
- psutil 5.9.7
- requests 2.31.0（新闻收集）
- gunicorn 21.2.0（生产环境）

## 注意事项
1. app.py文件不能有null字节，否则Railway部署失败
2. 中文字符在Windows PowerShell终端中会显示为乱码，但浏览器中正常显示
3. is_relative_to在Python 3.7中不可用，已替换为startswith
4. 水印CSS需要在每个页面模板中定义
5. 操作日志需要在每个操作中调用add_log函数记录
