## 任务执行规范

### 任务开始
- 任务开始时，必须先说"开始"，再执行任务

### 任务进行中
- 任务执行过程中，需要实时显示进度
- 使用进度条显示百分比（如：`[████████░░] 80%`）
- 进度更新频率：每完成一个关键步骤或每 10-20% 更新一次

### 任务中断
- 任务执行中途需要进行选择时，提醒用户"请进行选择"
- 任务执行中途需要中断时，提醒用户"任务已中断"

### 任务完成
- 任务完成后，必须说"完成"，进行结束
- 如果不在主界面，需要提醒用户返回主界面

## CLI 交互界面提示翻译

当遇到以下英文界面提示时，必须使用中文进行回应和回答：

| 英文原句 | 中文翻译 |
|---------|---------|
| Bash command | Bash 命令 |
| This command requires approval | 此命令需要授权确认 |
| Do you want to proceed | 是否继续执行 |
| Esc to cancel | 按 Esc 取消 |
| Tab to amend | 按 Tab 修改 |
| Ctrl+E to explain | 按 Ctrl+E 查看说明 |
